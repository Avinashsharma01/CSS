Units

There are 7 units in CSS:

  • em
  • ex
  • px
  • cm
  • mm
  • in
  • pt
  • pc

There are two types of units:

  • Absolute Units
  • Relative Units

Absolute Units

These units are fixed and do not change with the size of the browser window. They are used when the physical properties of the output medium are known. For example, if you are designing a web page for a printer, you can use absolute units.

  • cm: centimeters
  • mm: millimeters
  • in: inches
  • px: pixels
  • pt: points
  • pc: picas

Relative Units

These units are relative to another length property. For example, em is relative to the font-size of the element, and % is relative to the parent element.

  • em: Relative to the font-size of the element
  • ex: Relative to the x-height of the current font
  • %: Relative to the parent element
