# CSS Practice Repository

This repository is a collection of **HTML/CSS practice work** and small experiments.  
It is useful for learning core front-end styling concepts by reading and running example files directly in a browser.

## What this repo contains

- **Core CSS learning examples** in folders like `learn/`, `Grid/`, `Layoutt/`, and `Animation/`
- **Responsive design and media-query practice** in `Responsiveness/` and `media_quary/`
- **DOM and UI experiments** in `DOM/`, `Experiment/`, and other feature folders
- **Mini practice projects/clones** in folders such as `bootstrap/`, `flipcart/`, and `whatsapp/`

## What you can do with this repo

- Learn CSS topics (Flexbox, Grid, positioning, transitions, transforms, selectors, units)
- Open any `.html` file in your browser to see how the CSS works
- Modify styles and instantly observe visual behavior changes
- Reuse snippets and layout ideas in your own projects
- Use it as a personal sandbox for practicing front-end fundamentals

## How to run

No build setup is required. This is a static HTML/CSS repository.

1. Open the repository in VS Code (or any editor).
2. Open an HTML file directly in the browser **or** run with Live Server.
3. Explore folders and examples based on the topic you want to practice.
